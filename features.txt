
- add/+ ""
    
    adds "" as a task

- add/+ "" +a +b

    adds "" as a task with a and b as tags

- add/+ "" -4

    adds "" to pos 4 for current list

- ls +a

    lists all tasks with "a" tag

- ls

    lists all tasks

- ls +

    lists all tags

- mv 4 1

    moves task in position 4 to position 1 in current list
